Cách tạo:
1. Tạo Google Sheet mới với 2 sheet tab:
  • Sheet "ads" — cột: date | spend
  • Sheet "costs" — cột: collection | cost
2. File → Share → Publish to web → chọn "Entire document" + "CSV" → Publish
3. Copy link và dán vào đây
